# raylib web template
 
Minimal raylib (6.1) template for web.
 
## Requirements
 
- [Emscripten](https://emscripten.org/docs/getting_started/downloads.html)
- `make`
- `python3` (for local deploy)

## Setup

Set install path for EMSDK in Makefile, and you are good to go.

## Quick Start

```bash
./run.sh
```
